# PROJECT OVERVIEW

## TSRAONE Gateway

TSRAONE Gateway is the luminous root-domain portal for the TSRAONE digital universe.

This project is intentionally positioned as a flagship interaction and visual system rather than a conventional homepage. Its job is to transform the root domain into a premium spatial entry experience that routes users toward the broader subdomain network while simultaneously demonstrating a high standard of front-end craft.

## Active Domain Map

The current domain network presented by the portal is:

- `tsraone.com`
- `test2.tsraone.com`
- `test.tsraone.com`
- `aitrader.tsraone.com`
- `connect.tsraone.com`
- `note.tsraone.com`

## Core Purpose

The project serves two functions:

1. A production-facing root portal for the TSRAONE ecosystem.
2. A flagship GitHub showcase for material design, motion systems, interaction engineering, and modular front-end architecture.

## Product Goal

The intended emotional result is:

> standing inside a bright futuristic glass architecture under real sunlight

The site should feel:

- sunlit
- airy
- expensive
- spatial
- physically responsive
- controlled and elegant

It should not feel:

- cyberpunk
- dashboard-like
- neon-heavy
- template-based
- gimmicky

## Experience Strategy

The homepage is built around one dominant portal scene instead of a stack of marketing sections.

Key ingredients:

- a large central portal core
- six config-driven subdomain nodes
- directional sunlight and atmosphere
- premium glass material language
- a coherent physical interaction model

Supporting information is intentionally restrained so the world itself carries the first impression.

## Architectural Strategy

The codebase is organized as a system rather than a one-off composition. Visual concerns, content data, motion behavior, lighting logic, and portal rendering are split into dedicated modules so the project can evolve without collapsing into a single-file demo.

## Current Build Direction

The current rebuild focuses on:

- stronger hero-space composition
- improved sunlight hierarchy
- more refined glass surfaces
- more believable spatial depth
- better subdomain portal presentation
- more disciplined motion behavior

## Long-Term Direction

Future expansion should deepen the same core philosophy rather than replace it. The next gains should come from stronger material nuance, more spatial richness, and more elegant interaction polish, not from adding more panels or louder effects.
